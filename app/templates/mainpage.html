Welcome to StochSS! To get started, head over to the Model Editor and create new deterministic or well-mixed stochastic models for simulation. Or, if time is short, feel free to try out a model in the Example Library. Once you have defined your model, go to the Simulation Manager to perform your experiments, and then to the Status Page to view the output.


Do you want to run simulations in the Cloud?

StochSS will let you launch computational nodes in the Amazon EC2 compute cloud and use them to run your simulations, all with the click of a button! To use this feature, you need to sign up for an account with Amazon Web Services and then configure StochSS with your AWS credentials.


For more information, consult the StochSS Documentation. Also, make sure to join the StochSS Users Google group to receive information about updates and to ask questions.
