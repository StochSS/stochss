Welcome to StochSS Stochastic Simulation Service

Welcome to StochSS! In this first version, we provide a Model Editor to create new well-mixed stochastic models, or to import existing StochKit2 models in its native XML format. Once you have defined your model, you can run simulations with StochKit2.


Want to run simulations in the Cloud?

StochSS will let you launch computational nodes in the Amazon EC2 compute cloud and use them to run your simulations, all with the click of a button! To use this feature, you need to sign up for an account with Amazon Web Services and then configure StochSS with your AWS credentials.


For more information, consult the User Manual. Also, make sure to join the StochSS Users Google group to receive information about updates and to ask questions.
